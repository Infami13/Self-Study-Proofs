# Exercise 39      

Suppose to the contrary that $|ab| = m$. Given $|a| = n$ for finite $n$, then $a^n = e$. We the have

$$ e = e^n = ((ab)^m)^n = (ab)^{mn} = a^{mn}b^{mn} = a^{nm}b^{mn} = (a^n)^mb^{mn} = e^mb^{mn} = eb^{mn} = b^{mn},$$

for integer $mn$, which is clearly an absurdity as $b$ has infinite order. Note that the ability of $a$ and $b$ to commute allows us to claim $(ab)^{mn} = a^{mn}b^{mn}$.
